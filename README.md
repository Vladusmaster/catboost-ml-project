# catboost-ml-project
# Bank Reviews Sentiment Analysis (CatBoost)

Проект посвящён анализу текстовых отзывов о банках и построению модели
машинного обучения для определения тональности (положительный / отрицательный отзыв).

Модель построена с использованием алгоритма градиентного бустинга CatBoost,
который хорошо работает с текстовыми признаками и табличными данными.

## Цель проекта

Автоматически классифицировать тональность банковских отзывов
на основе текстового содержания.

## Используемые технологии

- Python
- CatBoost
- Pandas
- NumPy
- Scikit-learn
- TensorFlow / Keras (для работы с текстовыми данными)
- Matplotlib
- Jupyter Notebook

## Датасет

Датасет содержит текстовые отзывы клиентов банков.

Основные поля:

*Text* — текст отзыва
*Score* — тональность отзыва (Positive / Negative)

## Этапы проекта

1. Загрузка и первичный анализ данных
2. Очистка текста:
   - удаление email
   - удаление ссылок
   - удаление специальных символов
3. Подготовка признаков
4. Разделение выборки на обучающую и тестовую
5. Обучение модели CatBoost
6. Оценка качества модели

## Обучение модели

Модель обучалась с параметрами:

*iterations:* 500
*eval_metric:* Accuracy
*use_best_model:* True

Лучший результат на валидационной выборке:

Accuracy ≈ 0.95

## Результаты

Модель успешно определяет тональность банковских отзывов
с точностью около 95% на тестовой выборке.
